Symptom

  • The user is unable to view their asset balance on the Avalanche network when using the Metamask extension, despite having a positive balance.

Technical Root Cause

  • The error occurs due to a compatibility issue between the Metamask extension and the Avalanche network, resulting in the extension failing to fetch and display the asset balance.

Manual Overrides

  • Attempt to reset the Metamask extension by clearing its storage and restarting the browser.
  • Manually add the Avalanche network to the Metamask extension using the following configuration:
{
  "chainId": 43114,
  "chainName": "Avalanche",
  "nativeCurrency": {
    "name": "Avalanche",
    "symbol": "AVAX",
    "decimals": 18
  },
  "rpcUrls": ["https://api.avax.network/ext/bc/C/rpc"],
  "blockExplorerUrls": ["https://cchain.explorer.avax.network"]
}
  • Try updating the Metamask extension to the latest version.

You may want to try these commands in a separate terminal for more control.

Prerequisites

  • Metamask extension installed and configured on the browser.
  • Avalanche network added to the Metamask extension.
  • A positive asset balance on the Avalanche network.
