  • The error occurs due to the Geth upgrade not properly handling the existing blockchain data, resulting in a gap in the chain, which prevents the node from fully synchronizing with the Ethereum network.

Manual Overrides

  • Verify the Geth configuration to ensure it is set to sync the entire blockchain:
  • Check the geth command-line flags for the --syncmode option.
  • Run the command geth --syncmode "full" to enable full synchronization.
  • Use the following JSON configuration to override the default settings:

  "syncmode": "full"
}
