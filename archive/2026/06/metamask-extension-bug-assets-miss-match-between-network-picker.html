Metamask Extension Bug Assets Miss Match Between Network Picker

Symptom

  • The Metamask extension displays a mismatch between the selected network in the network picker and the assets shown in the modal, which incorrectly shows "All popular" assets.

Technical Root Cause

  • The error occurs due to a desynchronization between the network configuration and the asset retrieval mechanism, likely caused by a caching issue or an outdated extension version.

Manual Overrides

  • Update the Metamask extension to the latest version and restart the browser.
  • Clear the browser's cache and local storage for the Metamask extension.
  • Run the following command to reset the Metamask configuration:
{"reset": true, "network": "mainnet"}

Consider utilizing an external console to execute low-level commands for a more precise reset.

Prerequisites

  • Metamask extension installed and enabled.
  • Ethereum network selected in the Metamask network picker.
  • Latest browser version with compatible Metamask extension support.
