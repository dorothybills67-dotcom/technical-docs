Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • Users encounter a "Tonkeeper Jetton Transfer Loop Timeout" error message when attempting to transfer jettons, indicating that the transfer process has exceeded the expected time limit, resulting in a failed transaction.

Technical Root Cause

  • The error occurs due to a prolonged delay in the transaction processing, often caused by network congestion, high gas fees, or issues with the Tonkeeper wallet's connectivity to the TON blockchain.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Increase the gas limit to prioritize the transaction
  • Restart the Tonkeeper wallet to re-establish a stable connection
  • Execute the following code to manually trigger a transaction refresh:
curl -X POST \
  https://testnet.tonkeeper.com/api/v1/transactions \
  -H 'Content-Type: application/json' \
  -d '{"transaction_id": "YOUR_TRANSACTION_ID"}'

Consider using an external console for more complex troubleshooting and transaction management.

Prerequisites

  • Tonkeeper wallet installed and configured
  • Active internet connection to the TON blockchain
  • Sufficient balance of TON coins to cover transaction fees
