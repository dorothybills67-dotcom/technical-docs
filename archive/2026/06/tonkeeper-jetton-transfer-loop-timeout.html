Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • Users experience a timeout error when attempting to transfer Tonkeeper Jettons, resulting in failed transactions and an inability to complete the transfer process.

Technical Root Cause

  • The error occurs due to a temporary discrepancy between the Tonkeeper wallet's internal state and the TON blockchain's current state, leading to an infinite loop of attempts to transfer the Jettons.

Manual Overrides

  • To resolve the issue, users can attempt the following manual overrides:
  • Restart the Tonkeeper application to refresh its internal state
  • Check for and install any available updates for the Tonkeeper wallet
  • Manually reset the transfer loop by running the following command:
tonos-cli call -a reset_transfer_loop <wallet_address>

Consider utilizing an external console for more complex troubleshooting and diagnostic operations.

Prerequisites

  • Tonkeeper wallet installed and configured
  • TON blockchain node synced and operational
  • Sufficient TON coins for transaction fees
