Symptom

  • The user encounters a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer jettons, indicating that the transfer process has exceeded the expected time frame, resulting in a failed transaction.

Technical Root Cause

  • This error occurs due to a prolonged delay in the confirmation of the transaction on the TON blockchain, which can be caused by network congestion, high gas fees, or issues with the Tonkeeper wallet.

Manual Overrides

  • To resolve the issue, try the following:
  • Increase the gas limit to prioritize the transaction
  • Reduce the transaction amount to minimize network fees
  • Use an alternative wallet or node to bypass potential Tonkeeper issues
