Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • Users experience a timeout error when attempting to transfer Tonkeeper Jettons, resulting in failed transactions and a corresponding error message indicating a transfer loop timeout.

Technical Root Cause

  • The error occurs due to a temporary mismatch between the expected and actual transaction processing times within the TON blockchain, causing the transfer loop to exceed its predefined timeout threshold.

Manual Overrides

  • To resolve the issue, users can attempt the following manual overrides:
  • Increase the transaction processing timeout threshold
  • Manually retry the transfer operation
  • Utilize an alternative transfer method, such as using a different wallet or exchange
