Symptom

  • Users experience a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer jettons within the TON ecosystem, resulting in failed transactions and frustration.

Technical Root Cause

  • The error occurs due to a timeout in the transfer loop, which can be caused by network congestion, high gas fees, or incorrect configuration of the Tonkeeper wallet.

Manual Overrides

  • To resolve the issue, users can try the following manual overrides:
  • Increase the gas limit to prioritize the transaction
  • Reduce the transaction amount to decrease the load on the network
  • Update the Tonkeeper wallet to the latest version to ensure compatibility with the TON ecosystem

  }
}

Prerequisites
