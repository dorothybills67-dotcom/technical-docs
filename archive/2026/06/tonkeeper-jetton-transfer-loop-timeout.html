Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • Users encounter a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer Jettons, resulting in failed transactions and potential loss of funds.

Technical Root Cause

  • The error occurs due to a timeout in the Tonkeeper Jetton transfer loop, which can be caused by network congestion, high gas fees, or issues with the Tonkeeper wallet configuration.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Increase the gas limit for the transaction
  • Reduce the transfer amount to avoid network congestion
  • Use a custom configuration to bypass the transfer loop timeout
{
  "config": {
    "gasLimit": 1000000,
    "transferTimeout": 30000
  }
}

Consider utilizing an external console for more advanced troubleshooting and configuration adjustments. Prerequisites: * Tonkeeper wallet installed and configured * Basic understanding of TON blockchain and gas fees * Access to the Tonkeeper wallet configuration files
