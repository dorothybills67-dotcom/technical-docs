Symptom

  • The Tonkeeper user interface displays a "Tonkeeper Jetton transfer loop timeout" error message, indicating that the jetton transfer process has exceeded the expected time limit.

Technical Root Cause

  • The error occurs due to a timeout in the Tonkeeper jetton transfer loop, which can be caused by network congestion, high gas fees, or issues with the Ton blockchain.

Manual Overrides

  • Increase the gas limit for the transaction to prioritize it in the network:
  • Use the --gas-limit flag when sending transactions
  • Adjust the gasPrice in the transaction settings
  • Modify the transaction configuration using the following code block:
tonos-cli call -6 0:xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx JettonTransfer \
  --gas-limit 200000 \
  --gas-price 1 \
  --payload '({"dest":"0:xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx","value":100,"payload":""})'

Consider utilizing an external console for advanced troubleshooting and configuration.

Prerequisites

  • Tonos-cli installed and configured on the system
  • A basic understanding of Ton blockchain and Tonkeeper
  • Sufficient TON coins for gas fees
