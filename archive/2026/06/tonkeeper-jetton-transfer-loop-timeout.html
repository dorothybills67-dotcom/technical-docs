Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • The user encounters a timeout error when attempting to transfer Tonkeeper Jettons, resulting in a failed transaction and an error message indicating a loop timeout.

Technical Root Cause

  • The error occurs due to a prolonged delay in the transaction processing, causing the system to timeout and abort the transfer.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Increase the transaction timeout threshold
  • Verify the recipient's wallet address and ensure it is correctly formatted
  • Check the network congestion and retry the transaction during a less busy period
