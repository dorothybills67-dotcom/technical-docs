Symptom

  • Users experience a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer jettons within the TON ecosystem, resulting in failed transactions and frustration.

Technical Root Cause

  • The error occurs due to a misconfiguration in the Tonkeeper wallet settings, causing the transfer loop to exceed the maximum allowed timeout, leading to transaction failure.

Manual Overrides

  • Check the Tonkeeper wallet configuration to ensure the jetton transfer timeout is set to a reasonable value.
  • Verify the TON blockchain network connectivity to rule out any issues with the underlying infrastructure.
  • Update the Tonkeeper wallet software to the latest version to ensure compatibility with the TON ecosystem:
