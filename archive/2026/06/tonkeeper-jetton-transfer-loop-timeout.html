Symptom

  • Users of the TON ecosystem may encounter an error message indicating a "Tonkeeper Jetton transfer loop timeout" when attempting to transfer jettons, which are a type of token within the ecosystem. This error typically manifests as a timeout or failure to complete the transfer process.

Technical Root Cause

  • The Tonkeeper Jetton transfer loop timeout error occurs due to issues with the synchronization or communication between the user's wallet, the Tonkeeper platform, and the TON blockchain. This can be caused by network congestion, wallet configuration problems, or temporary outages on the TON network.

Manual Overrides

  • To resolve the issue, users can try the following steps:
  • Check the TON blockchain status for any ongoing maintenance or issues that might be affecting the transfer process.
  • Verify that the wallet is properly configured and connected to the correct network.
  • Attempt to restart the transfer process after ensuring the wallet and network are functioning correctly.
