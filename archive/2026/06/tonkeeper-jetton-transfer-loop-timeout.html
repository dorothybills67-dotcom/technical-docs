Symptom

  • Users experience a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer jettons within the TON ecosystem, resulting in failed transactions and potential loss of funds.

Technical Root Cause

  • The error occurs due to a synchronization issue between the Tonkeeper wallet and the TON blockchain, causing the transfer loop to timeout and fail to complete.

Manual Overrides

  • Check the TON blockchain for any ongoing maintenance or issues that may be causing the synchronization problem.
  • Verify that the Tonkeeper wallet is updated to the latest version and configured correctly.
  • Use the following command to reset the Tonkeeper wallet's cache:
tonos-cli config reset

Consider utilizing an external console for advanced troubleshooting and configuration adjustments.

Prerequisites

  • Tonkeeper wallet installed and configured on the user's device
  • TON blockchain wallet and account set up
  • Basic understanding of TON ecosystem and jetton transfers
