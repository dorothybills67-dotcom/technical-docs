Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • Users experience a timeout error when attempting to transfer Tonkeeper Jettons, resulting in failed transactions and an inability to complete the transfer process.

Technical Root Cause

  • The error occurs due to excessive loop iterations during the transfer process, causing the system to timeout before the transaction can be confirmed on the TON blockchain.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Increase the transaction timeout threshold to accommodate longer processing times.
  • Implement a retry mechanism to re-initiate the transfer process in case of a timeout.
  • Adjust the gas price to prioritize the transaction and reduce processing time.
