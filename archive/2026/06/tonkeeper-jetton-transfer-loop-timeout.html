# Symptom

Symptom

  • The user encounters a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer jettons, indicating a failure in the transaction process due to a timeout in the loop that handles the transfer.

Technical Root Cause

  • This error occurs due to the Tonkeeper wallet's inability to complete the jetton transfer within the allocated time frame, potentially caused by network congestion, insufficient gas, or issues with the smart contract governing the jetton transfer.

Manual Overrides

  • Increase the gas limit for the transaction to ensure it has enough resources to complete.
  • Adjust the transaction timeout settings in the Tonkeeper wallet configuration to allow for more time to process the transfer.
tonos-cli config set timeout 300

Consider utilizing an external console for more advanced troubleshooting and configuration adjustments.

Prerequisites

  • Ensure the latest version of Tonos-cli is installed.
  • Verify that the Tonkeeper wallet is properly configured and connected to the TON network.
  • Check for any ongoing network maintenance or issues that could be affecting transaction processing.
