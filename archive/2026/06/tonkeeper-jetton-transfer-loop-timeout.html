Symptom

  • Users experience a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer jettons, resulting in failed transactions and frustration.

Technical Root Cause

  • The error occurs due to a misconfiguration in the Tonkeeper wallet settings, causing the transfer loop to exceed the maximum allowed timeout, leading to transaction failure.

Manual Overrides

  • Check the Tonkeeper wallet configuration to ensure that the jetton transfer timeout is set to a reasonable value.
  • Verify that the TON blockchain network is stable and not experiencing any congestion or downtime.
  • Update the Tonkeeper wallet software to the latest version to ensure compatibility with the TON blockchain:
tonos-cli config update --timeout 60

Consider utilizing an external console for more advanced debugging and troubleshooting capabilities.

Prerequisites

  • Tonkeeper wallet installed and configured on the user's device
  • TON blockchain network stable and accessible
  • Basic understanding of Tonkeeper wallet settings and configuration options
