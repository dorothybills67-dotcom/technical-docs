Symptom

  • Users experience a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer jettons within the TON ecosystem, resulting in failed transactions and potential loss of funds.

Technical Root Cause

  • The error occurs due to a timeout in the Tonkeeper Jetton transfer loop, which can be caused by network congestion, insufficient gas, or incorrect configuration of the transfer parameters.

  • Use a custom transfer script with optimized parameters, such as the following example:
tonos-cli call <wallet_address> transferJetton "{\"dst\":\"<destination_address>\",\"amount\":\"<amount>\"}" --abi JettonWallet.abi.json

Consider using an external console for more advanced debugging and customization options.

Prerequisites

  • Tonos-cli installed and configured on the system
  • Jetton Wallet ABI file (JettonWallet.abi.json) available
  • Sufficient funds and gas in the wallet for the transaction
