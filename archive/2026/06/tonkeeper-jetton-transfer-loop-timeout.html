Symptom

  • Users experience a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer jettons within the TON ecosystem, resulting in failed transactions and frustration.

Technical Root Cause

  • The error occurs due to a timeout in the Tonkeeper Jetton transfer loop, often caused by network congestion, incorrect configuration, or insufficient gas for the transaction.

Manual Overrides

  • To resolve the issue, try the following:
  • Increase the transaction timeout value to allow for longer processing times.
  • Verify that the Tonkeeper wallet is properly configured and synced with the TON network.
  • Adjust the gas price to ensure sufficient funds for the transaction, as shown in the following example:
{
  "transaction": {
    "gas_price": 1000000000,
    "gas_limit": 100000
  }
}

Consider utilizing an external console for more advanced troubleshooting and configuration adjustments.

Prerequisites

  • Tonkeeper wallet installed and configured
  • Basic understanding of TON network and transaction mechanics
  • Sufficient TON coins for transaction fees
