Metamask Extension Bug Notifications Notification Badge Red Num

Symptom

  • The Metamask extension icon displays a red notification badge with a number, indicating unread notifications, even after the user has read and cleared all notifications.

Technical Root Cause

  • The error occurs due to a synchronization issue between the Metamask extension and the Ethereum blockchain, causing the notification badge to malfunction and display incorrect information.

Manual Overrides

  • Clear browser cache and restart the browser to reset the Metamask extension state.
  • Update the Metamask extension to the latest version using the following command:
npm install -g @metamask/core
  • Reset the Metamask notification badge manually by accessing the extension's internal settings.

Prerequisites: * Metamask extension installed and enabled * Ethereum blockchain connection established * Browser cache and cookies cleared
