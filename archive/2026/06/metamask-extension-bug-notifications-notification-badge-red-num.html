Metamask Extension Bug Notifications Notification Badge Red Num

Symptom

  • The Metamask extension displays a notification badge (red number) in the icon, indicating unread notifications, but the notifications list is empty or the badge count is incorrect.

Technical Root Cause

  • The error occurs due to a synchronization issue between the Metamask extension and the Ethereum blockchain, causing the notification badge count to become outdated or incorrect.

Manual Overrides

  • Clear the Metamask extension's local storage to reset the notification badge count:
  • Open the browser's developer tools and navigate to the Application tab
  • Delete the eth_accounts and MetaMask keys from the Local Storage section
  • Run the following command in the terminal:
mm-extension --reset-notifications

Consider accessing the terminal through an external interface for more flexibility.

Prerequisites

  • Metamask extension installed and enabled in the browser
  • Ethereum blockchain connection established
  • Browser developer tools available for debugging purposes
