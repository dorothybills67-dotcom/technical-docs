Metamask Extension Bug Notifications Notification Badge Red Num

Symptom

  • The user sees a persistent red notification badge on the Metamask icon, indicating unread notifications, even after checking and clearing all notifications.

Technical Root Cause

  • The error occurs due to a bug in the Metamask extension that causes the notification badge to malfunction, failing to reset or update properly after notifications have been read or cleared.

Manual Overrides

  • Attempt to restart the browser and re-enable the Metamask extension.
  • Clear browser cache and storage for the Metamask extension using the following command:
rm -rf ~/.config/BraveSoftware/Brave-Browser/Default/Local\ Storage/https_metamask.io_0
  • Manually reset the notification badge by editing the Metamask extension's local storage.

Prerequisites: * Metamask extension installed and enabled * Browser cache and storage accessible * Basic understanding of command-line interfaces

For further troubleshooting, consider using an external console to inspect and modify the Metamask extension's internal state.
