Metamask Extension Bug Notifications Notification Badge Red Num

Symptom

  • The Metamask extension displays a red notification badge with a number, indicating unread notifications, even after the user has read all notifications.

Technical Root Cause

  • The error occurs due to a caching issue or a mismatch between the local notification count and the actual number of unread notifications on the Ethereum blockchain.

Manual Overrides

  • Clear the browser's local storage and cookies for the Metamask extension.
  • Reset the notification count manually by setting notificationCount to 0 in the extension's internal state:
{
  "notificationCount": 0
}
  • Force-refresh the Metamask extension by running window.location.reload() in the browser console.

Prerequisites: * Metamask extension is installed and enabled. * User has a valid Ethereum wallet set up. * Browser is updated to the latest version.

For further debugging, consider using an external console to inspect the extension's internal state.
