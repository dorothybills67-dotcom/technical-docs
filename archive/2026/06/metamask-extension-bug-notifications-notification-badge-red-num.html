Metamask Extension Bug Notifications Notification Badge Red Num

Symptom

  • The user sees a red notification badge on the Metamask extension icon, indicating unread notifications, but the notifications list appears empty or the badge number does not match the actual number of unread notifications.

Technical Root Cause

  • The error occurs due to a mismatch between the local notification cache and the actual notification data stored on the Ethereum blockchain or Metamask's internal notification system, often caused by a failed synchronization or outdated extension version.

Manual Overrides

  • Clear local storage and cookies for the Metamask extension
  • Disable and re-enable the Metamask extension to force a resync
  • Update the Metamask extension to the latest version and run the following command:
mm-extension --reset-notifications

Consider utilizing an external console for more complex debugging tasks.

Prerequisites

  • Metamask extension installed and configured
  • Ethereum wallet set up and connected
  • Latest version of the browser and operating system
