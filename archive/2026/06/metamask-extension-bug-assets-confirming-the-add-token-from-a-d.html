Metamask Extension Bug Assets Confirming The Add Token From A D

Symptom

  • Users encounter a persistent "Add token" page when attempting to confirm the addition of a token from a decentralized application (dapp) using the Metamask extension, with the dialog box failing to complete the confirmation process.

Technical Root Cause

  • The error occurs due to a bug within the Metamask extension, specifically related to the handling of token addition requests from dapps, resulting in an infinite loop or a failure to properly confirm the token addition.

Manual Overrides

  • To resolve the issue, attempt the following manual overrides:
  • Clear the browser cache and restart the browser
  • Disable and re-enable the Metamask extension
  • Execute the following JSON-RPC request:
{
  "method": "eth_accounts",
  "params": [],
  "id": 1,
  "jsonrpc": "2.0"
}

Consider utilizing an external console to execute the JSON-RPC request for more flexibility.

Prerequisites

  • Metamask extension installed and configured
  • Ethereum dapp with token addition functionality
  • Basic understanding of JSON-RPC requests and Ethereum ecosystem
