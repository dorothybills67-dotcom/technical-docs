Technical Root Cause

Manual Overrides

  • Update the import statement to use the Tag component instead of Chip:
  • Replace import { Chip } from '@material-ui/core'; with import { Tag } from '@component-library/tag';
  • Update the component usage to match the new Tag component API
  • Remove any deprecated Chip component props to ensure compatibility
npm install @component-library/tag
