Symptom

  • The Metamask extension is displaying an error message indicating that the Chip component is deprecated and needs to be replaced with the Tag component from the component-library.

Technical Root Cause

  • The error occurs because the Chip component is no longer supported and has been replaced by the Tag component in the component-library, which is a part of the Web3 ecosystem, specifically designed for Ethereum (ETH) based applications.

Manual Overrides

  • Update the import statement to point to the new Tag component:
  • Replace import { Chip } from '@material-ui/core'; with import { Tag } from '@component-library/tag';
  • Update the component usage to use the new Tag component
  • Verify that the component-library is up-to-date
npm install @component-library/tag@latest

Consider using an external console to verify the package installation.

Prerequisites

  • Node.js installed on the system
  • npm package manager installed
  • Metamask extension installed and configured
