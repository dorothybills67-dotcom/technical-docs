Symptom

  • The user encounters an error message indicating that the Chip component is deprecated and needs to be replaced with the Tag component from the component-library in the Metamask extension.

Technical Root Cause

  • The error occurs because the Chip component is no longer supported and has been replaced by the Tag component, which offers improved functionality and compatibility with the latest Ethereum (ETH) ecosystem updates.

Manual Overrides

  • Update the import statement to use the Tag component:
  • Replace import { Chip } from '@material-ui/core'; with import { Tag } from '@component-library/tag';
  • Modify the component usage to match the new Tag component API
  • Verify that all dependencies are updated to the latest version
npm install @component-library/tag@latest

Consider utilizing an external console for further debugging and exploration of the component-library documentation.

Prerequisites

  • Node.js (version 16 or higher) installed on the system
  • npm (version 8 or higher) installed on the system
  • Metamask extension source code accessed and editable
