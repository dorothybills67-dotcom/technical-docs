Metamask Extension Bug Perps Deposit And Convert Musd This Payment Route Isn't Available Error

Symptom

  • Users encounter an error message stating "This payment route isn't available" when attempting to deposit and convert mUSD using the Metamask extension for Perps.

Technical Root Cause

  • The error occurs due to a temporary issue with the payment routing mechanism within the Metamask extension, which is unable to establish a viable payment path for the requested transaction.

Manual Overrides

  • Attempt to restart the Metamask extension and retry the transaction.
  • Check for updates to the Metamask extension and ensure you are running the latest version.
  • Utilize the following JSON configuration to manually override the payment routing:
{
  "paymentRoute": "manual",
  "routeConfig": {
    "mUSD": {
      "conversionRate": 1.0,
      "fee": 0.01
    }
  }
}

Consider utilizing an external console to diagnose and resolve the issue.

Prerequisites

  • Latest version of the Ethereum blockchain client
  • Sufficient ETH balance for transaction fees
