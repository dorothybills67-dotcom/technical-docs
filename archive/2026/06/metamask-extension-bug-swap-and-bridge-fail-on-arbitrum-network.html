Metamask Extension Bug Swap And Bridge Fail On Arbitrum Network

Symptom

  • Users experience failed transactions when attempting to swap or bridge assets on the Arbitrum network using the Metamask extension, resulting in error messages indicating a failure to complete the transaction.

Technical Root Cause

  • The error occurs due to a compatibility issue between the Metamask extension and the Arbitrum network, specifically related to the handling of transaction payloads and gas limits.

  • Increase the gas limit manually for the transaction: eth_estimateGas

}
  • Update the Metamask extension to the latest version to ensure compatibility with the Arbitrum network
