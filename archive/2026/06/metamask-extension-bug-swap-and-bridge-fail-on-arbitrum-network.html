Metamask Extension Bug Swap And Bridge Fail On Arbitrum Network

Symptom

  • Users experience a failure when attempting to swap or bridge assets on the Arbitrum network using the Metamask extension, resulting in an error message indicating the transaction cannot be completed.

Technical Root Cause

  • The error occurs due to a compatibility issue between the Metamask extension and the Arbitrum network, likely stemming from outdated configuration files or incorrect network settings.

Manual Overrides

  • Update the Metamask extension to the latest version to ensure compatibility with the Arbitrum network.
  • Reset the Metamask network settings to default values.
