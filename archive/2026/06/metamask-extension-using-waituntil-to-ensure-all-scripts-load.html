Metamask Extension Using Waituntil To Ensure All Scripts Load

Symptom

  • Users experience delayed or failed loading of web pages when using the Metamask extension, resulting in a poor browsing experience.

Technical Root Cause

  • The Metamask extension relies on the waitUntil function to ensure that all necessary scripts are loaded before allowing the webpage to proceed, but this function can sometimes cause issues if not implemented correctly.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Check the Metamask extension version and update to the latest version if necessary.
  • Verify that all scripts are properly loaded by checking the browser console for any errors.
  • Use the following code block to override the waitUntil function:
{
  "content_scripts": [
    {
      "matches": ["*://*/*"],
      "js": ["contentScript.js"],
      "run_at": "document_end"
    }
  ]
}

Consider using an external console to further investigate the issue.
