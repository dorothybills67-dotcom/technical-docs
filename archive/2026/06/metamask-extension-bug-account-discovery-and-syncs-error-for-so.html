Metamask Extension Bug Account Discovery And Syncs Error For Social Login Import Users

Symptom

  • Users attempting to import their accounts via social login encounter an error, resulting in failed account discovery and synchronization, preventing access to their Ethereum (ETH) wallets within the MetaMask extension.

Technical Root Cause

  • The error occurs due to a misconfiguration or incompatibility between the social login platform's API and the MetaMask extension, leading to a failure in retrieving and syncing user account data.

Manual Overrides

  • To resolve the issue, try the following steps:
  • Restart the browser and retry the social login import process.
  • Clear the MetaMask extension's storage and cache:
{
  "reset": true,
  "clearCache": true
}
  • Manually configure the API endpoints for social login import in the MetaMask settings. For a more in-depth analysis, consider utilizing an external console for direct debugging.

Prerequisites

  • MetaMask extension installed and updated to the latest version.
  • Social login platform account properly configured and connected.
  • Ethereum wallet created and accessible outside of the MetaMask extension.
