Symptom

  • Users encounter a "TronLink bandwidth error" when attempting to perform transactions or interact with smart contracts on the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to process transactions and execute smart contracts on the network.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens for a specified period.
  • Adjust the transaction's bandwidth and energy limits in the TronLink settings.
  • Use the following JSON-RPC command to query the account's bandwidth and energy:
{
  "jsonrpc": "2.1",
  "method": "getaccount",
  "params": [
    {
      "address": "user_account_address"
    }
  ],
  "id": 1
}

Consider utilizing an external console for more advanced debugging and customization options.

Prerequisites

  • A TRON account with sufficient TRX balance.
  • TronLink wallet extension installed and configured.
  • Basic understanding of TRON's bandwidth and energy mechanisms.
