Symptom

  • Users encounter a "TronLink bandwidth error" when attempting to perform transactions or interact with smart contracts on the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to process transactions and execute smart contract functions on the network.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens for a specified period.
  • Adjust the transaction's bandwidth and energy limits in the TronLink settings.
  • Manually set the bandwidth parameters using the following code block:
tronweb.tx.createTransaction(
  "your_address",
  {
    to: "recipient_address",
    value: 0,
    feeLimit: 100000000,
    bandwidthLimit: 100000000
  }
)

Consider utilizing an external console for more complex transactions.

Prerequisites

  • A TRON account with a sufficient balance of TRX tokens.
  • The latest version of TronLink installed.
  • A basic understanding of TRON's bandwidth and energy system.
