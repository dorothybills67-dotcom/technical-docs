Tronlink Bandwidth Error

Symptom

  • Users encounter an error message indicating a bandwidth issue when attempting to perform transactions or interact with smart contracts on the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to process transactions and execute smart contract functions on the network.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens for a specified period.
  • Adjust the transaction's bandwidth and energy limits in the TronLink settings.
  • Use the following command to check the account's current bandwidth and energy:
{
  "jsonrpc": "2.0",
  "method": "getaccount",
  "params": [
    {
      "address": "user's_tron_address"
    }
  ],
  "id": 1
}

Consider utilizing an external console for more advanced troubleshooting and direct access to the TRON network's API.

Prerequisites

  • A TRON account with a positive balance.
  • The latest version of TronLink installed.
  • A basic understanding of the TRON network and its resource management system.
