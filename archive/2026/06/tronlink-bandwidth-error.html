Tronlink Bandwidth Error

Symptom

  • Users encounter an error message indicating a bandwidth issue when attempting to perform transactions or interact with the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to process transactions and smart contract interactions on the network.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens, which can be done through the TronLink interface or using the following commands:
  • Freeze 10000 TRX for bandwidth
  • Unfreeze TRX after the bandwidth freeze period expires
  • Check the current bandwidth and energy of the account using:
trongrid account <account_address>

Consider utilizing an external console for more complex operations.

Prerequisites

  • TRX tokens available for freezing
  • TronLink installed and set up
  • Basic understanding of the TRON network and its resources (bandwidth and energy)
