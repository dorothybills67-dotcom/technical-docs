Tronlink Bandwidth Error

Symptom

  • Users encounter an error message stating "Tronlink bandwidth error" when attempting to perform transactions or interact with smart contracts on the TRON network using the TronLink wallet.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to process transactions and execute smart contracts on the network.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens for a specified period.
  • Adjust the wallet's settings to optimize energy consumption and allocation.
  • Use the following JSON-RPC command to manually override the bandwidth settings:
{
  "jsonrpc": "2.0",
  "method": "wallet/easyTransfer",
  "params": [
    {
      "from": "your_account_address",
      "to": "recipient_address",
      "amount": 100,
      "visible": false,
      "token_id": "1000000"
    }
  ],
  "id": 1
}

Consider utilizing an external console for more complex adjustments.

Prerequisites

  • TronLink wallet installed and set up
  • Sufficient TRX tokens in the wallet for bandwidth allocation
  • Basic understanding of TRON network and transaction mechanics
