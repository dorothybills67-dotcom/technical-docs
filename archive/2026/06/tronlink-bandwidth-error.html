Symptom

  • Users encounter a "TronLink bandwidth error" when attempting to perform transactions or interact with smart contracts on the TRON network using the TronLink wallet.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to process transactions and execute smart contracts on the network.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens for a specified period.
  • Allocate additional energy to the account by staking TRX tokens.
  • Adjust the wallet's settings to optimize bandwidth and energy consumption:
{
  "bandwidth": {
    "freeze": {
      "amount": 1000,
      "duration": 3
    }
  }
}

Consider utilizing an external console for further troubleshooting and configuration.

Prerequisites

  • A TRON account with a sufficient TRX balance
  • The latest version of the TronLink wallet installed
  • A basic understanding of TRON's bandwidth and energy mechanisms
