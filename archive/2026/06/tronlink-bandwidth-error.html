Tronlink Bandwidth Error

Symptom

  • Users encounter an error message indicating a bandwidth issue while attempting to conduct transactions or interact with smart contracts on the TRON network via TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth on the user's TRON account, which is required for transaction processing and smart contract execution on the network.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens, which then allocates bandwidth based on the amount frozen.
  • Adjust the bandwidth parameters manually through the TRON protocol's API, if directly interacting with the blockchain.
  • Use the following command to check the current bandwidth:
trongrid account get <account_address>

Consider utilizing an external console for more complex adjustments and better control over your TRON account settings.

Prerequisites

  • A TRON account with TronLink installed
  • Basic understanding of TRON's bandwidth and energy system
  • Sufficient TRX tokens for freezing to allocate bandwidth
