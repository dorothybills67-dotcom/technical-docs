Tronlink Bandwidth Error

Symptom

  • Users encounter an error message indicating a TronLink bandwidth issue, preventing them from completing transactions or interacting with the TRON network.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to perform transactions on the network.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens for a specified period.
  • Adjust the transaction's bandwidth and energy limits using the following JSON configuration:
{
  "bandwidthLimit": 10000,
  "energyLimit": 100000
}
  • Utilize the TRON Web API to manually trigger a bandwidth update using the updateAccount method. Consider executing these commands in a more flexible environment, such as an external console.

Prerequisites

  • A TRON account with sufficient TRX tokens
  • The latest version of TronLink installed
  • A basic understanding of TRON's bandwidth and energy mechanisms
