Tronlink Bandwidth Error

Symptom

  • Users encounter an error message indicating a TronLink bandwidth issue, preventing them from completing transactions on the TRON network. The error message may appear when attempting to send TRX, tokens, or interact with smart contracts.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to process transactions on the network. This can happen when the account's bandwidth or energy is depleted, often due to recent transaction activity.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX for bandwidth:
  • Freeze a minimum of 100 TRX to obtain additional bandwidth
  • Use the TRON wallet to freeze TRX and obtain bandwidth
  • Configure the following JSON to update the account's bandwidth parameters:
{
  "address": "user_tron_address",
  "frozen_balance": 100,
  "frozen_duration": 3
}

Consider utilizing an external console to execute these commands for more flexibility.

Prerequisites

  • A TRON wallet with a sufficient TRX balance
  • Basic understanding of TRON's bandwidth and energy system
  • Recent transaction history to identify potential bandwidth depletion causes
