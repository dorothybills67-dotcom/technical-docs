Tronlink Bandwidth Error

Symptom

  • Users encounter an error message "TronLink bandwidth error" when attempting to perform transactions or interact with smart contracts on the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to process transactions and execute smart contracts.

Manual Overrides

  • Increase the account bandwidth by freezing TRX tokens for bandwidth.
  • Check and adjust the transaction parameters to optimize energy consumption.
  • Use the following JSON-RPC command to query the account's bandwidth:
{
  "jsonrpc": "2.0",
  "method": "tron_getAccount",
  "params": [
    {
      "address": "your_tron_address"
    }
  ],
  "id": 1
}

Consider using an external console for more advanced troubleshooting and manual overrides.

Prerequisites

  • A TRON account with sufficient TRX balance.
  • TronLink extension installed and configured.
  • Basic understanding of TRON network and transaction mechanics.
