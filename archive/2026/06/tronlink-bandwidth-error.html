Symptom

  • Users encounter a "TronLink bandwidth error" when attempting to perform transactions or interact with the TRON network using the TronLink wallet.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's account, which is required to process transactions on the TRON network. This is a security measure to prevent spam transactions and ensure network stability.

Manual Overrides

  • Increase account bandwidth by freezing TRX tokens for a specified period.
  • Adjust transaction parameters to optimize energy consumption.
  • Utilize the TronLink API to monitor and manage account resources:
curl -X POST \
  https://api.trongrid.io/wallet/getaccount \
  -H 'Content-Type: application/json' \
  -d '{"address": "your_tron_address"}'

Consider utilizing an external console for more complex diagnostics and overrides.

Prerequisites

  • A basic understanding of the TRON network and its resources (TRX, bandwidth, and energy)
  • A TronLink wallet with a registered account
  • Familiarity with command-line interfaces and JSON formatting
