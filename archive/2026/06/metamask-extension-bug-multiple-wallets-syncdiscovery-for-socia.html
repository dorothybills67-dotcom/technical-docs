Metamask Extension Bug Multiple Wallets Syncdiscovery For Social

Symptom

  • Users experience issues with Metamask extension during the Social Import onboarding process, where multiple wallets fail to sync or discover properly, resulting in an error message indicating a bug in the Metamask extension.

Technical Root Cause

  • The error occurs due to a conflict between the wallet's synchronization mechanism and the Social Import feature, which attempts to import and sync multiple wallets simultaneously, overwhelming the extension's handling capabilities.

Manual Overrides

  • Attempt to restart the Metamask extension and retry the Social Import process.
  • Manually remove and re-add the affected wallets to resolve any potential configuration issues.
  • Utilize the following command to reset the wallet's synchronization state:
mm-sync-reset --wallet-id=<wallet_id> --network=ethereum

Consider utilizing an external console for further troubleshooting.

Prerequisites

  • Metamask extension installed and enabled
  • Ethereum network selected
  • Social Import feature enabled
