Metamask Extension Bug Ledger If Blind Signing Is Disabled Reco

Symptom

  • Users encounter an issue with the Metamask extension when using a Ledger device, where the extension fails to function properly if blind signing is disabled, resulting in a requirement to reconnect from the beginning of the workflow.

Technical Root Cause

  • The error occurs due to a compatibility issue between the Metamask extension and the Ledger device when blind signing is disabled, causing a disruption in the expected workflow.

Manual Overrides

  • To resolve the issue, users can attempt the following manual overrides:
  • Enable blind signing on the Ledger device to restore compatibility with the Metamask extension.
  • Update the Metamask extension to the latest version, which may include fixes for the compatibility issue.
  • Use the following JSON configuration to override the default settings:
{
  "ledger": {
    "blindSigning": true
  }
}

Consider utilizing an external console for further debugging and configuration.

Prerequisites

  • Metamask extension installed and configured
  • Ledger device connected and set up
  • Blind signing disabled on the Ledger device
