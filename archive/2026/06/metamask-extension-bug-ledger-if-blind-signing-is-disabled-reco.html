Metamask Extension Bug Ledger If Blind Signing Is Disabled Reco

Symptom

  • The user encounters an error when attempting to connect their Ledger device to Metamask, specifically when blind signing is disabled, resulting in a "Reconnect from the beginning" workflow failure.

Technical Root Cause

  • The error occurs due to a bug in the Metamask extension that handles communication with Ledger devices, particularly when blind signing is disabled, leading to an inability to establish a secure connection.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Update Metamask to the latest version to ensure the bug is patched.
  • Reset the Ledger device and retry the connection process.
  • Use the following command to clear the Metamask cache:
localStorage.clear('metamask')
