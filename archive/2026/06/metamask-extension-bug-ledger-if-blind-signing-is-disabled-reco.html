Metamask Extension Bug Ledger If Blind Signing Is Disabled Reco

Symptom

  • Users encounter an error when attempting to connect their Ledger device to Metamask, specifically when blind signing is disabled, resulting in a prompt to reconnect from the beginning of the workflow.

Technical Root Cause

  • The error occurs due to a compatibility issue between Metamask's extension and the Ledger device's configuration, particularly when blind signing is disabled, leading to an incomplete or failed connection process.

Manual Overrides

  • Try enabling blind signing on the Ledger device to see if it resolves the issue.
  • Use the following command to reset the Metamask extension's connection settings:
mm-extension reset --ledger
  • Clear the browser's cache and storage for the Metamask extension.

Prerequisites: * Metamask extension installed and updated * Ledger device properly configured and connected * Blind signing disabled on the Ledger device

Consider utilizing an external console to troubleshoot and resolve the issue.
