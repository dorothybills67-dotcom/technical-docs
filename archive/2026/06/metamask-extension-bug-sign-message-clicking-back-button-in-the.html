Metamask Extension Bug Sign Message Clicking Back Button In The

Symptom

  • Users experience a bug when using the Metamask extension to sign a message, specifically when clicking the back button in the Bitcoin Snap Sign Message confirmation dialog. The expected behavior is to return to the previous screen, but instead, the extension becomes unresponsive or displays an error message.

Technical Root Cause

  • The error occurs due to a mismatch between the expected state of the extension and the actual state after clicking the back button. This mismatch causes the extension to fail to handle the sign message request correctly, resulting in the bug.

Manual Overrides

  • To troubleshoot the issue, try the following manual overrides:
  • Check the Metamask extension version and update to the latest version if necessary.
  • Clear the browser cache and restart the browser.
  • Use the following JSON configuration to reset the sign message settings:
{
  "signMessage": {
    "bitcoinSnap": {
      "backButtonBehavior": "cancel"
    }
  }
}

Consider utilizing an external console for more advanced debugging.
