Metamask Extension Bug Sign Message Clicking Back Button In The

Symptom

  • When using the Metamask extension to sign a message, clicking the back button in the Bitcoin Snap Sign Message confirmation dialog causes the extension to malfunction, resulting in an error.

Technical Root Cause

  • The error occurs due to a timing issue between the Metamask extension and the Bitcoin Snap Sign Message confirmation dialog. When the back button is clicked, the extension is not properly notified, leading to a state mismatch and subsequent error.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Clear the Metamask extension's cache and storage
  • Reset the Bitcoin Snap Sign Message confirmation dialog settings
  • Execute the following command to reset the extension's state:
mm-reset --sign-message-state

Consider using an external console to execute these commands for a more seamless experience.

Prerequisites

  • Metamask extension installed and enabled
  • Bitcoin Snap Sign Message confirmation dialog configured
  • Latest version of the Metamask extension and Bitcoin Snap plugin installed
