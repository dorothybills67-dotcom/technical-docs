Metamask Extension Bug Sign Message Clicking Back Button In The

Symptom

  • Users experience a bug when using the Metamask extension to sign a message, specifically when clicking the back button in the Bitcoin Snap Sign Message confirmation dialog. The error causes the extension to malfunction, resulting in a failed transaction or an inability to complete the signing process.

Technical Root Cause

  • The error occurs due to a compatibility issue between the Metamask extension and the Bitcoin Snap Sign Message protocol. When the back button is clicked, the extension fails to properly handle the cancellation of the signing process, leading to a corrupted transaction state.

Manual Overrides

  • To resolve the issue, users can try the following manual overrides:
  • Update the Metamask extension to the latest version
  • Clear the browser cache and cookies
  • Run the following command in the terminal:
metamask --reset --ethereum-rpc-url https://mainnet.infura.io/v3/YOUR_PROJECT_ID

Consider using an external console for more advanced troubleshooting.

Prerequisites

  • Metamask extension installed and configured
  • Bitcoin Snap Sign Message protocol supported
  • Latest version of the browser and operating system
