Metamask Extension Bug Unable To Cancel Pending Transaction Due

Symptom

  • Users are unable to cancel pending transactions in Metamask, resulting in an error message stating "not enough gas" despite having sufficient funds.

Technical Root Cause

  • The error occurs due to a bug in the Metamask extension that incorrectly estimates the gas required for the transaction, leading to a failed cancellation attempt.

Manual Overrides

  • Increase the gas limit manually using the following commands:
  • eth_estimateGas to estimate the correct gas limit
  • eth_sendTransaction with the corrected gas limit
  • eth_cancelPendingTransaction with the updated transaction parameters

  ],

}
