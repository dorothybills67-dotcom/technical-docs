Symptom

  • When sending an NFT using the Metamask extension, the activity display shows the NFT's ID as if it were the amount being sent, instead of displaying the actual amount (e.g., 1 NFT).

Technical Root Cause

  • This error occurs due to a bug in the Metamask extension's UI rendering, where it incorrectly maps the NFT's ID to the amount field, causing the display mismatch.
