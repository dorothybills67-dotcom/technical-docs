Symptom

  • The Java Tron node fails to sync with the TRON network, resulting in a failure to retrieve or send transactions, and displays a "java-tron [Bug]" error message.

Technical Root Cause

  • The error occurs due to a compatibility issue between the Java Tron node and the TRON network, often caused by an outdated node version or incorrect configuration.

Manual Overrides

  • Update the Java Tron node to the latest version to ensure compatibility with the TRON network.
  • Verify that the node configuration files are correctly set up, including the config.conf file.
  • Use the following command to reset the node and re-sync with the network:
java -jar java-tron.jar --reset

Consider using an external console for more advanced troubleshooting and configuration options.

Prerequisites

  • Java Tron node installed and configured on the system
  • TRON network connection established
  • Latest Java version installed and compatible with the Java Tron node
