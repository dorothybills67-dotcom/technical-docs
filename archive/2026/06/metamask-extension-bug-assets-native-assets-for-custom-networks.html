Metamask Extension Bug Assets Native Assets For Custom Networks

Symptom

  • Users are unable to view native assets for custom networks when selected in the Metamask extension, resulting in an incomplete display of available assets.

Technical Root Cause

  • The error occurs due to a bug in the Metamask extension's asset handling mechanism, which fails to properly fetch and display native assets for custom networks.

Manual Overrides

  • To resolve the issue, users can attempt the following manual overrides:
  • Update the Metamask extension to the latest version
  • Manually add the native asset token for the custom network using the following JSON configuration:
{
  "chainId": "0x1234",
  "tokenAddress": "0x5678",
  "tokenSymbol": "CUSTOM",
  "tokenDecimal": 18
}
  • Clear the Metamask extension's cache and restart the browser

Prerequisites: * Metamask extension installed and configured * Custom network added to the Metamask extension * Native asset token information available for the custom network For further troubleshooting, consider using an external console to inspect the Metamask extension's logs and configuration.
