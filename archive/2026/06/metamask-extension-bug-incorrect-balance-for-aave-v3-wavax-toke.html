Metamask Extension Bug Incorrect Balance For Aave V3 Wavax Token

Symptom

  • Users of the Metamask extension on the Avalanche network may notice an incorrect balance displayed for their Aave v3 WAVAX tokens, potentially leading to confusion and incorrect financial decisions.

Technical Root Cause

  • The issue arises from a discrepancy in the way Metamask handles token balances for Aave v3 protocols on the Avalanche network, likely due to an outdated or incorrect configuration.

Manual Overrides

  • Update Metamask to the latest version to ensure compatibility with the latest Aave v3 and Avalanche network updates.
  • Reset the Metamask token cache by deleting the tokenCache.json file, which can be done by running the following command in the terminal:
rm ~/.metamask/tokenCache.json
  • Manually add the Aave v3 WAVAX token to Metamask using the token's contract address and decimal places.

Prerequisites: * Metamask extension installed and configured on the Avalanche network * Aave v3 account set up with WAVAX tokens * Basic understanding of blockchain and cryptocurrency concepts
