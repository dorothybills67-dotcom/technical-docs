Metamask Nonce Out Of Sync

Symptom

  • Users encounter an error message stating "MetaMask nonce out of sync" when attempting to send transactions or interact with smart contracts on the Ethereum network.

Technical Root Cause

  • The error occurs due to a mismatch between the expected transaction nonce in MetaMask and the actual nonce on the Ethereum blockchain, often resulting from concurrent transactions or failed transaction attempts.

Manual Overrides

  • To resolve the issue, users can try the following:
  • Reset the MetaMask account to resynchronize the nonce.
  • Manually adjust the nonce in the transaction settings to match the current block number.
  • Utilize the following JSON-RPC method to fetch the current nonce:
{
  "jsonrpc": "2.0",
  "method": "eth_getTransactionCount",
  "params": ["0x742d35Cc6634C0532925a3b844Bc454e4438f44e", "latest"],
  "id": 1
}

Consider using an external console for more complex queries.

Prerequisites

  • MetaMask extension installed and configured
  • Ethereum account with a positive balance
  • Stable internet connection
